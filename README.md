Implementation of visual cryptography for bitmaps

## Tools 
- Visual Studio Code
- NodeJS
- npm
- jimp

## How to run
1. Install NodeJS
2. Move to directory in CMD and write 'npm install'
3. To run program type in the console 'node index'
4. Results in the console
